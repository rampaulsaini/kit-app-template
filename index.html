Omniverse
Kit App Template

GPU-ACCELERATED · OPENUSD · RTX RENDERING · PHYSICAL AI · CLOUD STREAMING

⚡EngineOmniverse Kit SDK
🎮RenderingNVIDIA RTX
📦FormatOpenUSD
☁️StreamingWebRTC / GDN
🐍LanguagePython / C++
🌐DeployNGC / NVCF / AWS
A toolkit for building GPU-accelerated 3D applications within the NVIDIA Omniverse ecosystem. Create high-performance, OpenUSD-based desktop or cloud-streaming applications using the Omniverse Kit SDK — with RTX ray tracing, PhysX physics, and AI capabilities built-in.
▶ Quick Start 📦 Templates ⚙ GitHub ꙰ Shromani Platform
◈

Core Capabilities

Omniverse Kit SDK — Key Features

🔮
OpenUSD Foundation

Build on Pixar's Universal Scene Description — the industry standard for 3D data interchange. Create, manipulate and render rich 3D content across tools and pipelines.

⚡
RTX Rendering

NVIDIA RTX real-time ray tracing with physically accurate lighting, materials, and global illumination. Photorealistic visualization at interactive framerates.

☁️
Cloud Streaming

Stream Kit applications directly to web browsers via WebRTC. Deploy on NVIDIA Cloud Functions (NVCF), GDN, or Kubernetes clusters for global reach.

🤖
Physical AI

Integrate with NVIDIA's AI ecosystem — Isaac Sim, PhysX, Warp, and more. Build digital twins, robotics simulations, and AI-powered 3D applications.

🧩
Extension System

Modular architecture with Python and C++ extensions. Browse 200+ extensions from the Extension Manager. Build and publish your own to the community registry.

🏭
Enterprise Ready

Production-grade stability via NGC Catalog. Docker containerization, CI/CD tooling, packaging, and deployment workflows built-in from day one.

◈

Application Templates

5 Pre-Built Templates — Start Immediately

Each template is a fully configured Omniverse application — from minimal to feature-rich.

Kit Service
Headless Service
Minimal definition of an Omniverse Kit SDK based service. Useful for creating headless services leveraging Omniverse Kit functionality without a GUI.
Base Editor
Kit Base Editor
Minimal template for loading, manipulating and rendering OpenUSD content from a graphical interface. Ideal starting point for custom 3D editors.
USD Composer
Scene Authoring
Authoring complex OpenUSD scenes and configurators. Features Fabric Scene Delegate, AXF MDLs, Variant Tools, and layout/materials/lighting.
USD Explorer
Scene Explorer
Exploring and collaborating on large OpenUSD scenes. Built for teams reviewing complex 3D assets and digital twins in real time.
USD Viewer
Streaming Viewer
Viewport-only template with RTX rendering, app streaming, and messaging. Easily streamed to web browsers via WebRTC. Ideal for client-facing deployments.

Quick Start

Get Started in Minutes

Terminal — Omniverse Kit Setup
# Clone the repository
$ git clone https://github.com/rampaulsaini/kit-app-template.git
$ cd kit-app-template
 
# Create new application from template
$ ./repo.sh template new
? Select what you want to create: Application
? Select desired template: USD Viewer
? Enter application name: shromani_omniverse
 
# Build and launch
$ ./repo.sh build
$ ./repo.sh launch
✓ Application launched successfully
✓ RTX renderer initialized
✓ Streaming ready at localhost:8011
